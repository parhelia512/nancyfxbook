This page was requested at @Model.WhenRequested



This example uses before and after filters attached directly to the module serving this request.

If you observe the time this page was created when refreshing it you'll see the page is handled by an output cache, this cache has a sliding window of 30 seconds.

As long as your refreshing the page, you'll reset the timer and the cache will continue to wait 30 seconds after the last request before expiring, if you request the page then leave it for 30 seconds before re-requesting it, you'll see you get a new copy.
