Welcome to the NancyFX demo site

This site was built to demonstrate the capabilities provided by the NancyFX .NET web framework.

It was designed to accompany the book "NancyFX Succinctly" published by Syncfusion which can be downloaded for free at:

https://www.syncfusion.com/resources/techportal/details/ebooks/nancyfx



Please note, this demo app requires a fairly recent HTML5 capable browser as it uses some of the newer features found in the newer browsers

It may not run correctly in anything less than IE10 or Chrome/Firefox older than mid 2014
